# 电信行业 profile 立法文档（industry-profile-telecom）

> 版本：v1（2026-07-23，WP-V 批次）
> 条款纪律：条款号仅采用已核验事实；不虚构条款号、不虚构 URL——标准统一指向国家标准全文公开系统检索页或 3GPP/ISO 官方页，法律统一指向国家法律法规数据库，访问日期均 2026-07-23；行业共识但未经原文核实的条款号显式标注「条款号待原文核实」。
> 配套配置包：`assets/industry-profiles/telecom.conf`（用法：`cat` 追加到 `precheck.conf` 末尾后按项目裁剪）。
> 门禁基线：49 既有（`precheck.sh` GATE_FLAGS 注册表，含 WP-S1/S2 合规门禁）；本 profile 不新增门禁，仅覆盖配置开关。
> 边界声明：电信关基等保测评/密评/定级备案、电信设备进网许可属线下外审/行政许可，本范式不覆盖认证——门禁输出**不构成等保测评合规证据**（§3 差额项）。

## 0. 定位与适用

**适用对象**：基础电信企业/增值电信企业的核心网（5GC/EPC）、业务支撑系统（BOSS/CRM/计费）、网管系统（OMC/网优）、电信云/IDC 承载系统、物联网连接管理平台软件研发交付；为电信运营商提供产品/服务的信息技术供应商的研发交付亦属此列。落入《关键信息基础设施安全保护条例》范畴的电信行业关基系统同受约束。

**监管基线**：电信行业交付的合规底座是「等保 + 密评 + 个人信息保护 + 供应链」四条线——
1. **等级保护**：《网络安全法》第 21 条（等级保护制度）+ GB/T 22239-2019（等保 2.0 基本要求）；电信行业扩展：YD/T 1729-2008《电信网和互联网安全等级保护实施指南》（行业共识，条款号待原文核实）；电信关基系统原则上等保三级起步；
2. **密码合规**：《密码法》2020 +《商用密码管理条例》2023；GB/T 39786-2021 密评（等保三级及以上系统须开展商用密码应用安全性评估）；
3. **个人信息保护**：《电信和互联网用户个人信息保护规定》（工信部令第 24 号，2013）——电信业务经营者收集使用用户个人信息须遵循合法、正当、必要原则并取得用户同意；《个人信息保护法》2021 第 55-56 条（PIA 与记录）；电信业务直接处理海量用户个人信息（入网实名/通信详单/位置信息），属个人信息强监管；
4. **供应链安全**：GB/T 39204-2022 7.9（关基供应链）；3GPP TS 33.501《Security architecture and procedures for 5G System》（5G 安全架构，行业共识，条款号待原文核实）；YD/T 3627-2019《移动通信终端安全防护技术要求》（行业共识，条款号待原文核实）。

本 profile 把四条线落到门禁开关，差额项（等保测评/密评/定级备案/进网许可）显式登记人工核对，不做「门禁通过=合规通过」的越界承诺。

## 4. 配置包用法与姿态

`assets/industry-profiles/telecom.conf` 为 conf 片段：`cat assets/industry-profiles/telecom.conf >> precheck.conf`（bash 后赋值覆盖先赋值），再按项目裁剪——占位值 `<...>` 必须替换为真实路径。姿态约定：

- 未配置静默跳过；启用后 fail-closed（`--dengbao`/`--crypto`/`--privacy`/`--pia`/`--sbom`/`--oss-eval`/`--docs-pack`/`--release-sign`/`--rtm` 均属此列）；
- 豁免必须按 standards-compliance.md §F.2 五字段格式登记（对象|规则|理由|审批人|日期），空理由视为无效豁免 → fail；
- `DENGBAO_SCAN_DIRS`/`CRYPTO_SCAN_DIRS`/`PRIVACY_SCAN_DIRS` 为本 profile 必填裁剪项；`PRIVACY_SCAN_DIRS` 启用但配置目录全不存在 → fail（fail-closed 语义，切勿带占位值上生产）；
- 电信关基系统原则上 DENGBAO_LEVEL="3" 起步；确属二级系统的项目降级为 2 时须在项目文档说明定级依据（§3-① 备案材料为准）；
- 电信行业属个人信息强监管：PIA_REQUIRED=1（工信部令第 24 号 + 个保法第 55 条）；确无个人信息处理的纯网管子系统项目降为 0 时须在项目文档说明依据；
- 门禁输出**不构成** GB/T 28448-2019 等保测评合规证据（§3 差额项须线下外审）。
