## 勾稽关系（无多漏错重）

> 数据一致性核对项。

- [ ] 无遗漏：关联记录无缺失（如订单→明细无孤儿）
- [ ] 无多余：无冗余/重复记录
- [ ] 记录正确：字段值符合业务规则
- [ ] 勾稽正确：外键/聚合/关联关系正确
- [ ] 一致性：同源数据多处一致
- [ ] 幂等性：重复请求不产生副作用
